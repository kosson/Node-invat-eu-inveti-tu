# Aggregate.prototype.options

Conține opțiunile pasate comenzii aggregate. Opțiunile posibile sunt:

- `readPreference`
- `cursor`
- `explain`
- `allowDiskUse`
- `maxTimeMS`
- `bypassDocumentValidation`
- `raw`
- `promoteLongs`
- `promoteValues`
- `promoteBuffers`
- `collation`
- `comment`
- `session`
