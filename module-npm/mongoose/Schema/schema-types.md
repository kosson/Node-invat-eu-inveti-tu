# Schema.Types

Este o proprietate a clasei `Schema` cu ajutorul căreia poți defini tipul de date al datelor unei căi.

```javascript
var mongoose = require('mongoose');
var ObjectId = mongoose.Schema.Types.ObjectId;
```

## Tipurile

### `String`

### `Number`

### `Boolean` | Bool

### `Array`

### `Buffer`

### `Date`

### `ObjectId` | Oid

### `Mixed`
